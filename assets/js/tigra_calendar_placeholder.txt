<h2 class="first">Description</h2>
<img src="multiple-calendars.gif" alt="Calendars Preview" class="imgRight" />
<p>Tigra Calendar is a free cross-browser JavaScript Calendar control, it improves the user experience by enabling a drop-down calendar for date fields in web forms.</p>

<h2>Features</h2>

<link rel="stylesheet" type="text/css" href="tcal.css"> 
<script language="JavaScript" src="tcal.js"></script> 
<form style="float:right;clear:right;margin-top:10px;">LIVE SAMPLE: <input type="text" name="sampleField" class="tcal" /></form>

<ul>
	<li>Easy to use - no programming skills required</li>
	<li>Small footprint - no need to add large JavaScript framework if all you need is date picker</li>
	<li>Easy customization and styling via CSS</li>
	<li>Simple localization</li>
	<li>Attaches to existing form fields, plays nicely with existing styling</li>
	<li>Can be attached dynamically - perfect for AJAX applications</li>
	<li>Supports any date format similar to PHP's date(..) function</li>
	<li>Efficient, well formatted code full of code snippets to borrow - great learning source</li>
	<li>Works in all modern browsers including mobile platforms</li>
	<li>Can be used in any kind of web applications (PHP/ASP/CGI etc.)</li>
	<li>Public domain software - <span style="color:red">FREE</span> to use any way you want</li>
</ul>

<h2>Benefits</h2>

<ul>
	<li>Fast and simple date entry improves user experience</li>
	<li>Use of the calendar eliminates date formatting errors</li>
	<li>Invalid dates such as February 30 can not be entered via the calendar</li>
</ul>

<h2>Setup</h2>
<ol>
	<li>Upload calendar files to your website</li>
	<li>Link calendar resources (JS and CSS) to your pages</li>
	<li>Add class="tcal" to text input(s)</li>
</ol>

<h2><a href="/products/tigra_calendar_pro/">Tigra Calendar PRO</a> additionally offers:</h2>
<ul>
	<li>Even more appearance configuration options including the layout of the calendar controls</li>
	<li>Events and API can be used to hook multiple calendars for date range selection and implementing various business logic</li>
	<li>Date limits (minimum and maximum allowed dates) are supported as either absolute date or relatively in days from today</li>
	<li>Reset date to today button</li>
	<li>Forbidden dates list (by date, by day of week etc)</li>
	<li>List of holidays or other dates requiring special highlighting (for example those not available for booking) supported</li>
	<li>Allowed dates list (blacks out all the dates except those listed)</li>
	<li>Configurable behavior on out of range date selection</li>
	<li>Multiple modes: drop-down, pop-up and in-page calendars</li>
</ul>
